About Taskilon

TASKILON is a TODO app designed and developed by Arjun Thakur. It is a project to showcase some of my skills. The code for this app is available at Github. The front-end is completely "pure code"1 without any use of front-end frameworks. The core or the backend is developed using Python's Django Framework. The app contains o-auth signup facility, i.e. the user can signup using his/her social profile2 which is good, as the user doesn't have to remember so many passwords for different profiles. The app is currently being hosted on HEROKU, so the access might be slow. The core of the app will be lauched in the coming weeks. The app will be completely ready soon!
